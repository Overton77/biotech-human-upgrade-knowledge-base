# TruDiagnostic biomarker and report inventory

## Scope and completeness

This note consolidates the publicly exposed report outputs, biomarker names, analyte families, and metric families visible across TruDiagnostic marketing pages, provider pages, shop product pages, help-center pages, and sample-report references.

**Completeness label:** `substantially improved but still partial`

- High confidence for top-level report families and many named analytes.
- Medium confidence for exact total counts because different public pages conflict.
- Improved by a public sample TruAge PDF and support-center articles.
- Still incomplete for any metrics only visible inside customer portals, provider-only views, or a non-public TruHealth sample report.

## Public inconsistencies to preserve

- TruHealth appears as `105+`, `110+`, and in some public marketing contexts `150+` biomarkers.
- TruAge is described as `+75 key biomarkers`, but the report families mix direct analytes, inferred scores, fitness proxies, and risk-style models.
- Some pages label OMICmAge as `OMICAge`, which should be treated as a naming variant rather than a separately verified product.
- Support-center and sample-report surfaces improve clarity but still do not fully resolve the `TruAge` versus `TruAge COMPLETE` packaging distinction.
