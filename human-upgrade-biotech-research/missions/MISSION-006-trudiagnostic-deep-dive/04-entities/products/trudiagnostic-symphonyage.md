# SYMPHONYAge

## Summary

`SYMPHONYAge` is TruDiagnostic's organ-system aging report family, exposed publicly as an `11`-system aging score set bundled into TruAge.

## Publicly Exposed Details

- Public role: organ-systems age report within TruAge
- Publicly named systems:
  - Brain
  - Blood
  - Lungs
  - Heart
  - Liver
  - Hormone
  - Immune System
  - Musculoskeletal
  - Inflammation
  - Kidneys
  - Metabolic
- Public framing:
  - system-specific epigenetic aging algorithm
  - evaluates how organs age independently and synchronously
  - intended to support targeted health interventions

## Cautions

- Public pages expose the system list clearly, but not a detailed technical methods note for the current commercial implementation.
- Stage 3 should separate company commercial packaging from any underlying published systems-age methodology.
