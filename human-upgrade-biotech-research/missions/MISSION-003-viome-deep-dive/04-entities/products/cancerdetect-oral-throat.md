# CancerDetect Oral & Throat

## Product Details

| Field | Value |
|---|---|
| URL | `https://cancerdetect.viome.com/` |
| Current price captured | `from $599` |
| Sample type | saliva |
| Positioning | at-home diagnostic aid for oral and throat cancer risk detection |
| Turnaround | about `4 weeks` after provider review |

## Public Safety / Regulatory Framing

- explicitly says the test is `not FDA-approved or cleared`
- also says the program received FDA `Breakthrough Device` designation for accelerated review
- marketed as a lab-developed test performed in a `CLIA-certified` laboratory in Bothell, Washington
- provider review and consultation are built into the testing flow

## Public Performance Claims

- website claim: specificity `95%`
- website claim: OSCC sensitivity `90%`
- website claim: OPSCC sensitivity `90%`
- intended as a diagnostic aid, not a definitive diagnosis

## Evidence Stack

- `PMID 37454545`: peer-reviewed oral / throat cancer paper describing a machine-learning classifier from saliva host + microbial metatranscriptomic signatures
- `NCT05451303`: active observational registry for the OralViome Cancer Testing System
- `NCT06174428`: larger current validation registry targeting sensitivity, specificity, PPV, and NPV

## Assessment

CancerDetect is Viome's strongest evidence-linked commercial program in this mission. It still should be treated carefully: promising, actively validated, and more substantial than a generic wellness claim, but not yet an established standard-of-care screening tool.
